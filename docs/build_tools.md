## Node Version Manager -- nvm

windows version download
https://github.com/coreybutler/nvm-windows/releases

```bash 
# see what versions and now using in local
nvm list

# downlowd newest 
nvm install node
# use newest
nvm use newest

# download version 
nvm install <version>
# use specific version 
nvm use <version>

```
